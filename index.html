Estudo Tags h1 até h6

(Inicio)

A tag h é uma das tags HTML que define um cabeçalho. O número que segue a letra “h” indica o nível de importância do cabeçalho, variando de 1 a 6. O cabeçalho de nível 1 é o mais importante e o de nível 6 é o menos importante. A sintaxe para a tag -h- é -h1- Texto do cabeçalho -/h1-. O texto do cabeçalho é exibido em negrito e com um tamanho de fonte maior do que o texto normal da página 1

Estudo Tags p

(Inicio)

A tag p é uma das tags HTML que define um parágrafo. A sintaxe para a tag -p- é -p> Texto do parágrafo -/p>. O texto do parágrafo é exibido em um bloco separado do restante do conteúdo da página, com uma margem superior e inferior

Estudo Tags mark

(Inicio)

A tag HTML mark representa um trecho de destaque em um texto. Ela é utilizada para marcar um texto que deve ser destacado ou realçado, como uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular. Por exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque 1. A sintaxe para a tag -mark> é Texto a ser destacado

Estudo Tags small

(Inicio)

A tag small é utilizada para reduzir o tamanho da fonte num intervalo definido de um texto 2. Por exemplo , se você quiser que um texto apareça em tamanho menor do que o texto ao redor, você pode usar a tag -small> para isso. O texto dentro da tag -small> será exibido em um tamanho de fonte menor

Estudo Tags (i - u - strong)

(Inicio)

  1. Tag i

    2. A tag i é utilizada para colocar um texto em itálico . Por exemplo, se você quiser que um texto apareça em itálico, você pode usar a tag -i> para isso. O texto dentro da tag -i> será exibido em itálico .

  2. Tag u

    3. A tag u é um elemento de marcação de texto que é usado para adicionar ênfase a uma palavra ou frase. Quando o navegador encontra a tag u, ele sublinha o texto entre as tags. A tag u é uma tag de formatação de texto obsoleta e não deve ser usada em documentos HTML modernos. Em vez disso, você deve usar CSS para estilizar o texto.

  3. Tag strong

    4. A tag strong> é utilizada para definir um texto com forte importância. O conteúdo dentro dela é tipicamente exibido em negrito 2. A tag -b> pode ser usada para especificar texto em negrito sem nenhuma importância extra 2. A tag -strong> também suporta os atributos globais em HTML e os atributos de evento em HTML.

Estudo Tags (ol - ul - li)

(Inicio)

A tag ul em HTML é usada para criar listas não ordenadas. Ela é um elemento de marcação de texto que é usado para adicionar uma lista de itens sem ordem rígida. Para criar uma lista não ordenada, é necessário usar as tags ul e li , sendo a primeira usada para definir a lista e a segunda para cada item da lista. Os itens em uma lista desordenada são exibidos com um marcador que pode ter várias formas, como um ponto, um círculo ou um quadrado.

Estudo Tags (a e hr)

(Inicio)

A tag a em HTML é usada para criar um hyperlink. Ela é um elemento de marcação de texto que é usado para adicionar um link a uma página da web ou a um arquivo. Para criar um hyperlink, é necessário usar as tags a e href, sendo a primeira usada para definir o link e a segunda para especificar o URL de destino. O texto que aparece entre as tags a é o texto do link. A tag hr em HTML é usada para criar uma linha horizontal. Ela é um elemento de marcação de texto que é usado para adicionar uma linha horizontal na página. A tag hr é um elemento vazio, o que significa que não precisa de uma tag de fechamento. A tag hr é usada para separar o conteúdo da página em seções.

Estudo Tags sub

(Inicio)

A tag sub em HTML é usada para exibir texto abaixo da linha de base normal. Ela é frequentemente usada para exibir fórmulas químicas e matemáticas, como H2O e x2 1. Para usar a tag sub, basta envolver o texto que você deseja exibir abaixo da linha de base normal com as tags sub e /sub. Por exemplo, o código HTML A fórmula química para água é H 2 exibirá a fórmula química para água com o “2” abaixo da linha de base normal

Estudo Tags blockquote

(Inicio)

A tag blockquote é usada para definir uma seção de texto que é uma citação longa de outra fonte. Normalmente, o texto dentro da tag é indentado e possui um espaçamento adicional em relação ao texto normal, para indicar que é uma citação. O atributo cite pode ser usado para fornecer uma URL que designa um documento de origem ou mensagem para a informação citada. Para alterar o recuo de -blockquote>, use a propriedade CSS margin.
